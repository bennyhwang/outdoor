# 户外探险装备清单

一个功能完整的户外探险准备清单应用，支持徒步/自驾两种模式，自动推荐装备、计算重量、生成路线时间线和风险评估。

## 功能特性

- **双模式支持** — 徒步模式（计算背包重量）和自驾模式（无重量限制）
- **智能装备推荐** — 根据距离、地形、天气（温度/湿度/降雨）、过夜天数、海拔、难度自动适配
- **路线时间线** — 自动生成行程时间线
- **风险评估** — 自动识别风险并提供安全建议
- **背包重量计算** — 实时计算并检查负重上限
- **CSV 导出** — 一键导出装备清单
- **零依赖** — 单 HTML 文件，浏览器直接打开即可使用

## 快速开始

### 网页版（推荐）

直接用浏览器打开 `outdoor_checklist.html`，或在 Vercel 部署在线使用。

### Python 版

```bash
python offroad_checklist.py
```

## 使用说明

1. 填写行程信息（距离、地形、天气、人数、天数等）
2. 点击"生成装备"
3. 勾选需要携带的项目
4. 查看总重量和时间线
5. 导出 CSV 或截图保存
